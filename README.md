# Mumbai University Grievance Redressal Portal

Mumbai University Grievance Redressal Portal is a soon-to-be-used portal by the University of Mumbai to solve the grievances of the student. The Front End of the application is built using React while the backend is built using Node.js and Serverless Framework(AWS).
